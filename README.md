# Dashboard Exotic Carwash

## Overview

Dashboard Exotic Carwash is a web-based management system developed to support car wash business operations efficiently. The system provides an admin dashboard for managing transactions, customers, services, and operational data in one centralized platform.

This project was developed using Python Flask for the backend and relational database integration for data management.

---

## Features

* User Authentication & Session Login
* Admin Dashboard
* Customer Management
* Transaction Management
* Service Management
* Reporting System
* Responsive Web Interface
* Database Integration
* Business Operational Monitoring

---

## System Architecture

The application follows a web-based client-server architecture:

* Frontend handles user interaction and dashboard visualization.
* Flask backend processes business logic and API handling.
* PostgreSQL database stores customer, transaction, and operational data.
